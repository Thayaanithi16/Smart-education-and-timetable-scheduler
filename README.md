Smart Education and Timetable Scheduler

Overview

Smart Education and Timetable Scheduler is an AI-powered web application that helps students manage their academic activities efficiently. The system automatically generates personalized timetables, creates study plans, tracks assignments, schedules exams, and sends reminders to improve productivity and time management.

Features

- User Registration and Login
- Student Dashboard
- Smart Timetable Generation
- AI-Based Study Plan Generator
- Assignment Tracker
- Exam Scheduler
- Reminder Notifications
- Progress Analytics Dashboard
- Admin Panel

Technologies Used

- Frontend: React.js, Tailwind CSS
- Backend: FastAPI (Python)
- Database: MySQL
- Programming Languages: Python, JavaScript

Project Objectives

- Automate timetable creation.
- Generate personalized study plans.
- Track assignments and examinations.
- Improve student productivity through smart scheduling.
- Provide a user-friendly and responsive interface.

Installation

1. Clone the repository.
2. Install frontend and backend dependencies.
3. Configure the MySQL database.
4. Start the FastAPI backend server.
5. Run the React frontend.
6. Open the application in your web browser.
